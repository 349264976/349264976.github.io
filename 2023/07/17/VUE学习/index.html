VUE中的特点响应式显示

VUE的指令操作

V-HTML ：可以替换 document.innerHtml 可以解析html语句并且展示到标签内容

v-show

1.控制元素隐藏

2.语法：v-show=”表达式” 表达式的值 true显示 ，false 隐藏

实际上是控制标签的display：none来控制内容是否显示 是显示与不显示的问题

频繁操作的页面用

v-if

1.控制元素隐藏(条件渲染)

2.语法 v-if=”表达式“ 表达式值 true显示，false 隐藏

实际上是控制判断条件决定是否创建标签 是存在与不存在的问题

不经常操作的用v-if

v-else v-else-if

1.作用：辅助v-if 进行判断渲染

2.语法：v-else v-else-if=”表达式“

3.注意：需要紧挨着v-if一起使用

ag

               <div>
                <div v-if="score >= 90">优秀</div>
                <div v-else-if="score >= 70">普通</div>
                <div v-else-if="score >= 60">及格</div>
                <div v-else>成绩过低建议重开</div>
              </div>

v-on

1.作用：注册时间=添加监听+提供处理逻辑

2.语法：

1.v-on：事件名=”内敛语句”

v-on:==@

           <button v-on:click="count--">-</button> 
          <span>{{count}}</span>   
          <button v-on:click="count++">+</button>  
          <button @click="count--">-</button> 
          <span>{{count}}</span>   
          <button @click="count++">+</button>

2.v-on：时间名=”methods中的函数名字“

  </div>  
    <button @click="fn">显示or隐藏</button>
    <h1 v-show="flag">黑马程序员</h1>
    </div>
          methods: 
                 {
                    fn(){
                        this.flag=!this.flag;
                    }

                }

v-on调用传参数

v-bind：属性名=“表达式”

1.作用：动态的设置html的标签属性-》 src url title ….

2.语法： v-bind:属性名=“表达式”

可以将v-bind改写成：

    <img v-bind:src="imgurl" alt=""> tup</img>

v-for

v-for=”(item,index) in 数组”

item 每一项，index 下标

 <ul>
      <li v-for="(item, index) in booksList" 				                                    :key="item.id">
        <span>{{ item.name }}</span>
        <span>{{ item.author }}</span>
        <!-- 注册点击事件 →  通过 id 进行删除数组中的 对应项 -->
        <button @click="del(item.id)">删除</button>
      </li>
    </ul>


 <script>
    const app = new Vue({
      el: '#app',
      data: {
        booksList: [
          { id: 1, name: '《红楼梦》', author: '曹雪芹' },
          { id: 2, name: '《西游记》', author: '吴承恩' },
          { id: 3, name: '《水浒传》', author: '施耐庵' },
          { id: 4, name: '《三国演义》', author: '罗贯中' }
        ]
      },
      methods: {
        del (id) {
          // console.log('删除', id)
          // 通过 id 进行删除数组中的 对应项 → filter(不会改变原数组)
          // filter: 根据条件，保留满足条件的对应项，得到一个新数组。
          // console.log(this.booksList.filter(item => item.id !== id))
          this.booksList = this.booksList.filter(item => item.id !== id)
        }
      }
    })
  </script>

v-model

1.作用：给表单元素使用，双向的绑定 –>可以快速获取或者设置表单内容

v-model=”变量”

数据变化 视图自动更新

视图变化数据自动更新

双向绑定

 <div id="app">
    <!-- 
      v-model 可以让数据和视图，形成双向数据绑定
      (1) 数据变化，视图自动更新
      (2) 视图变化，数据自动更新
      可以快速[获取]或[设置]表单元素的内容
     -->
    账户：<input type="text" v-model="username"> <br><br>
    密码：<input type="password" v-model="password"> <br><br>
    <button @click="login">登录</button>
    <button @click="reset">重置</button>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
  <script>
    const app = new Vue({
      el: '#app',
      data: {
        username: '',
        password: ''
      },
      methods: {
        login () {
          console.log(this.username, this.password)
        },
        reset () {
          this.username = ''
          this.password = ''
        }
      }
    })
  </script>

<!-- 主体区域 -->
<section id="app">
  <!-- 输入框 -->
  <header class="header">
    <h1>小黑记事本</h1>
    <input v-model="todoName" placeholder="请输入任务" class="new-todo" />
    <button class="add" @click="add()">添加任务</button>
  </header>
  <!-- 列表区域 -->
  <section class="main">
    <ul class="todo-list">
      <li class="todo" v-for="(item,index) in list" v-bind:key="list">
        <div class="view">
          <span class="index">{{index+1}}.</span> <label>{{item.name}}</label>
          <button @click="del(item.id)" class="destroy">删除</button>
        </div>
      </li>
    </ul>
  </section>
  <!-- 统计和清空 -->
  <footer class="footer">
    <!-- 统计 -->
    <span class="todo-count">合 计:<strong> {{list.length}} </strong></span>
    <!-- 清空 -->
    <button v-show="list.length>0" class="clear-completed" @click="qk">
      清空任务
    </button>
  </footer>
</section>

<!-- 底部 -->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
<script>

  const app = new Vue({
    el: '#app',
    data: {
      list:[
        {id:1,name: "跑步一公里"},
        {id:2,name: "洗澡一小时"},
      ]
    },
  methods:{
    del(id){
      this.list=this.list.filter(item=>item.id !==id)
    },
    add(){
      if(this.todoName.trim()===""){
        alert("名称不能为空")
        return
      }
      this.list.unshift({
        id: +new Date(),
        name: this.todoName
      })
      this.todoName=""
    },
    qk(){
      this.list=[]
    }
  }
  })

</script>
</body>
</html>
