VUE的指令操作

V-HTML ：可以替换 document.innerHtml 可以解析html语句并且展示到标签内容

v-show

1.控制元素隐藏

2.语法：v-show=”表达式” 表达式的值 true显示 ，false 隐藏

实际上是控制标签的display：none来控制内容是否显示 是显示与不显示的问题

频繁操作的页面用

v-if

1.控制元素隐藏(条件渲染)

2.语法 v-if=”表达式“ 表达式值 true显示，false 隐藏

实际上是控制判断条件决定是否创建标签 是存在与不存在的问题

不经常操作的用v-if

v-else v-else-if

1.作用：辅助v-if 进行判断渲染

2.语法：v-else v-else-if=”表达式“

3.注意：需要紧挨着v-if一起使用

ag

               <div>
                <div v-if="score >= 90">优秀</div>
                <div v-else-if="score >= 70">普通</div>
                <div v-else-if="score >= 60">及格</div>
                <div v-else>成绩过低建议重开</div>
              </div>

v-on

1.作用：注册时间=添加监听+提供处理逻辑

2.语法：

1.v-on：事件名=”内敛语句”

v-on:==@

           <button v-on:click="count--">-</button> 
          <span>{{count}}</span>   
          <button v-on:click="count++">+</button>  
          <button @click="count--">-</button> 
          <span>{{count}}</span>   
          <button @click="count++">+</button>

2.v-on：时间名=”methods中的函数名字“

  </div>  
    <button @click="fn">显示or隐藏</button>
    <h1 v-show="flag">黑马程序员</h1>
    </div>
          methods: 
                 {
                    fn(){
                        this.flag=!this.flag;
                    }

                }

v-for

v-for=”(item,index) in 数组”

item 每一项，index 下标

 <ul>
      <li v-for="(item, index) in booksList" 				                                    :key="item.id">
        <span>{{ item.name }}</span>
        <span>{{ item.author }}</span>
        <!-- 注册点击事件 →  通过 id 进行删除数组中的 对应项 -->
        <button @click="del(item.id)">删除</button>
      </li>
    </ul>


 <script>
    const app = new Vue({
      el: '#app',
      data: {
        booksList: [
          { id: 1, name: '《红楼梦》', author: '曹雪芹' },
          { id: 2, name: '《西游记》', author: '吴承恩' },
          { id: 3, name: '《水浒传》', author: '施耐庵' },
          { id: 4, name: '《三国演义》', author: '罗贯中' }
        ]
      },
      methods: {
        del (id) {
          // console.log('删除', id)
          // 通过 id 进行删除数组中的 对应项 → filter(不会改变原数组)
          // filter: 根据条件，保留满足条件的对应项，得到一个新数组。
          // console.log(this.booksList.filter(item => item.id !== id))
          this.booksList = this.booksList.filter(item => item.id !== id)
        }
      }
    })
  </script>

v-model

1.作用：给表单元素使用，双向的绑定 –>可以快速获取或者设置表单内容

v-model=”变量”

数据变化 视图自动更新

视图变化数据自动更新

双向绑定

 <div id="app">
    <!-- 
      v-model 可以让数据和视图，形成双向数据绑定
      (1) 数据变化，视图自动更新
      (2) 视图变化，数据自动更新
      可以快速[获取]或[设置]表单元素的内容
     -->
    账户：<input type="text" v-model="username"> <br><br>
    密码：<input type="password" v-model="password"> <br><br>
    <button @click="login">登录</button>
    <button @click="reset">重置</button>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
  <script>
    const app = new Vue({
      el: '#app',
      data: {
        username: '',
        password: ''
      },
      methods: {
        login () {
          console.log(this.username, this.password)
        },
        reset () {
          this.username = ''
          this.password = ''
        }
      }
    })
  </script>

<script>

  const app = new Vue({
    el: '#app',
    data: {
      list:[
        {id:1,name: "跑步一公里"},
        {id:2,name: "洗澡一小时"},
      ]
    },
  methods:{
    del(id){
      this.list=this.list.filter(item=>item.id !==id)
    },
    add(){
      if(this.todoName.trim()===""){
        alert("名称不能为空")
        return
      }
      this.list.unshift({
        id: +new Date(),
        name: this.todoName
      })
      this.todoName=""
    },
    qk(){
      this.list=[]
    }
  }
  })

</script>

重要方法

数组.reduce

可以实现累加效果

return this.fruitList.reduce((sum, item) => sum + item, 0);

在这里，累加器函数 (sum, item) => sum + item 将每个元素依次加到累加器 sum 上，最终返回总和。初始值为 0，确保了当 fruitList 为空数组时，也能得到正确的结果。

array.find 实现找到指定id的数组元素

你的代码 fruit.num--; 是想要将找到的水果对象的 num 属性减 1。不过，在使用 find 方法找到对象后，需要确保对象存在才能执行减法操作，否则可能会导致错误。

const fruit = this.fruitList.find(item => item.id === id);
if (fruit) {
  fruit.num--;
}

通过添加条件判断，if (fruit) 确保了对象存在时才执行减法操作。这样可以避免在找不到对象时出现错误，确保代码的稳定性。

数组.filter 可以实现数组元素的删除

你的代码 this.fruitList = this.fruitList.filter(item => item.id !== id); 是想要从 fruitList 数组中移除指定 id 的水果对象。

filter 方法用于创建一个新数组，其中包含满足指定条件的所有元素。在这里，你可以使用 filter 方法来过滤掉指定 id 的水果对象。

以下是修正后的代码示例：

javascriptCopy Codethis.fruitList = this.fruitList.filter(item => item.id !== id);

通过使用 filter 方法并指定条件 item.id !== id，我们将筛选出 id 不等于给定值的水果对象，然后将结果赋值给 fruitList 数组。这样就实现了从数组中移除指定 id 的水果对象的操作。

VUE的生命周期

vue的生命周期函数（钩子函数）

 <script>
    const app = new Vue({
      el: '#app',
      data: {
        count: 100,
        title: '计数器'
      },
      // 1. 创建阶段（准备数据）
      beforeCreate () {
        console.log('beforeCreate 响应式数据准备好之前', this.count)
      },
      created () {
        console.log('created 响应式数据准备好之后', this.count)
        // this.数据名 = 请求回来的数据
        // 可以开始发送初始化渲染的请求了
      },

      // 2. 挂载阶段（渲染模板）
      beforeMount () {
        console.log('beforeMount 模板渲染之前', document.querySelector('h3').innerHTML)
      },
      mounted () {
        console.log('mounted 模板渲染之后', document.querySelector('h3').innerHTML)
        // 可以开始操作dom了
      },

      // 3. 更新阶段(修改数据 → 更新视图)
      beforeUpdate () {
        console.log('beforeUpdate 数据修改了，视图还没更新', document.querySelector('span').innerHTML)
      },
      updated () {
        console.log('updated 数据修改了，视图已经更新', document.querySelector('span').innerHTML)
      },

      // 4. 卸载阶段
      beforeDestroy () {
        console.log('beforeDestroy, 卸载前')
        console.log('清除掉一些Vue以外的资源占用，定时器，延时器...')
      },
      destroyed () {
        console.log('destroyed，卸载后')
      }
    })
  </script>

钩子函数 一般操作数据需要在created（）函数中进行
